Introduktion

Det här är Uppsala universitets mall för webbaserade system version 0.1. Mallen är ett hjälpmedel för systemutvecklare och webbdesigners vid nyutveckling eller underhåll av system och mallens syfte är att underlätta att våra system följer universitets grafiska riktlinjer samt skapa en enhetlig användarupplevelse för universitets alla användare.

Användning av mallen

Denna mall innehåller tre delar:

  • Filen index.html (denna fil) som ger förslag på övergripande html-struktur (se filens källkod).
  • En versionshanterad CSS som kan användas av alla system (se mappen css).
  • Mappen scss som används vid utveckling av mallens CSS.

Designsystemet

Denna mall följer Designsystemets riktlinjer men är bara en startpunkt för en bra användarupplevelse. Mer information om hur vi skapar en bra och enhetlig användarupplevelse hittar du i Designsystemet.

Senaste versionen av denna mall hittar du under Mall för nedladdning, och där hittar du också uppdaterad information om hur denna mall kan användas.
